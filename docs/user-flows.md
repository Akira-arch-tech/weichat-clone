# User Flows（用户流程）

## Flow 1：登录并进入聊天

```mermaid
flowchart TD
A[启动应用] --> B[登录页]
B --> C{输入账号密码}
C -->|成功| D[进入会话列表]
C -->|失败| E[提示错误信息]
D --> F[打开某个会话]
F --> G[发送文本消息]
G --> H[消息显示在聊天窗口]

Flow 2：通讯录加好友
flowchart TD
A[通讯录页] --> B[搜索用户]
B --> C[进入用户详情]
C --> D[发送好友申请]
D --> E[对方收到申请]
E --> F{同意?}
F -->|同意| G[双方成为好友]
F -->|拒绝| H[申请结束]

Flow 3：朋友圈发布与互动
flowchart TD
A[发现页] --> B[进入朋友圈]
B --> C[点击发布]
C --> D[输入文字/添加图片]
D --> E[发布成功]
E --> F[Feed 展示新动态]
F --> G[他人点赞]
F --> H[他人评论]

Flow 4：编辑个人资料
flowchart TD
A[我] --> B[设置]
B --> C[编辑资料]
C --> D[修改昵称/签名/地区]
D --> E[保存成功]
E --> F[个人信息更新]

页面状态建议（最小）
每个关键页面至少覆盖：

Loading（加载中）
Empty（空状态）
Error（失败态）
Success（成功态）
